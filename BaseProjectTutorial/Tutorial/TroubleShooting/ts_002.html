逆引き

実行後に「__super::???」を入れてくださいというようなエラーになる

　エラーになったタイミングで存在しているすべてのObjectの項目の???の関数に注目してください。

エラーになった時、Init, Update, Drawなどが??? に入っているはずです 

bool Init() 
{ 
    __super::Init(); //< この一行が抜けている可能性
    
    // 実際のInitの処理 
    return true;
}

void Update() 
{ 
    __super::Update(); //< この一行が抜けている可能性
    
    // 実際のUpdateの処理 
}

void Draw() 
{ 
    __super::Draw(); //< この一行が抜けている可能性
    
    // 実際のDrawの処理 
}
