Vector

Vectorとは

float2(二次元)や float3(三次元)で表され、 座標 や 方向 として利用できます

　座標として利用

■ プレイヤーの位置
■ アイテムの位置
■ 敵出現の位置 など

プログラムでの利用方法の例
X座標:100 Y座標:10 Z座標 1 に配置します 
obj->SetTranslate( { 100.0f, 10.0f, 1.0f } );

position(float3)に座標を取得します 
auto postion = obj->GetTranslate();

　方向として利用

■ プレイヤーの向き (BP関数では角度を利用しているためあまり利用しません)
■ 敵への方向 など

プログラムでの利用方法
敵の方向ベクトルの方向に進めます 
// 自分のポジション
auto my_postion    = my_object->GetTranslate();
// 敵のポジション
auto enemy_postion = enemy_object->GetTranslate();

// 敵の位置から自分の位置を引くと、vec に敵への方向ベクトル(正規化してないもの)が入ります
auto vec = enemy_position - my_position;

// 当然以下のようにすると自分の位置は、敵の位置に到着しますよね (いきなりそこに行ってほしくはないが・・・)
// my_position += vec;

// ということで その方向のまま、1 の長さにしてしまいます
// つまり方向も、float3の(X,Y,Z)の3要素で表現できることがわかると思います
vec = normalize( vec );

// その状態で例えば3.0のスピードで敵の方向に進めるのは以下となります
my_object->AddTranslate( 3.0f * vec );
